# Feature Engineering on Housing Dataset

## Project Overview

This project demonstrates Feature Engineering techniques on the California Housing Dataset using Pandas and Scikit-learn.

## Tools Used

- Python
- Pandas
- Scikit-learn

## Dataset

California Housing Dataset from Scikit-learn.

Features include:
- Median Income
- House Age
- Average Rooms
- Average Bedrooms
- Population
- Average Occupancy
- Latitude
- Longitude

No external dataset download is required.

## Objectives

- Create new features
- Apply One-Hot Encoding
- Scale numerical features
- Generate interaction features
- Prepare data for machine learning models

## Feature Engineering Techniques

### 1. One-Hot Encoding
Converted categorical income categories into numerical columns.

### 2. Feature Scaling
Standardized numerical features using StandardScaler.

### 3. Interaction Features
Created new features by combining existing variables:
- Rooms_Per_Person
- Income_HouseAge

## Workflow

1. Load California Housing Dataset.
2. Create categorical features.
3. Apply One-Hot Encoding.
4. Scale numerical variables.
5. Generate interaction features.
6. Create engineered dataset.

## Deliverables

- Engineered Dataset
- Python Code
- Feature Engineering Report

## Results

The dataset was transformed into a machine-learning-ready format through encoding, scaling, and feature creation techniques.

## Conclusion

Feature engineering enhances data quality and improves the effectiveness of machine learning models by creating more informative and useful features.
